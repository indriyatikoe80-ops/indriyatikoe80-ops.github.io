# 📋 PROJECT OVERVIEW
## Website Promosi Aplikasi Belajar Anak

---

## 🎯 Tujuan Proyek

Membuat website promosi single-page (landing page) yang **cantik, menarik, dan profesional** untuk mempromosikan 1 aplikasi edukasi Android untuk anak usia dini (TK/PAUD). Website ini harus mampu:

1. Menarik perhatian orang tua dan guru sebagai pengambil keputusan download
2. Menarik minat anak-anak dengan visual yang ceria dan berwarna
3. Memberikan informasi lengkap tentang fitur aplikasi
4. Memudahkan proses download dari Google Play Store
5. Membangun kepercayaan dengan menampilkan screenshot, testimoni, dan informasi developer

## 🧒 Target Audiens Website

Website ditargetkan kepada **2 audiens utama**:

### 1. Orang Tua (Decision Maker)
- Mencari aplikasi edukasi yang aman dan berkualitas untuk anak
- Ingin tahu fitur, keamanan konten, dan manfaat pembelajaran
- Terpengaruh oleh rating, ulasan, dan tampilan profesional

### 2. Guru TK/PAUD
- Mencari media belajar digital interaktif
- Ingin tahu konten materi dan kesesuaian kurikulum
- Membutuhkan bukti nyata berupa screenshot dan video demo

## 🛠️ Technology Stack

Website ini dibangun menggunakan stack modern untuk performa maksimal dan kemudahan pengembangan:

1.  **Framework Utama:** React.js (v18+)
2.  **Build Tool:** Vite (Untuk bundling ultra-cepat)
3.  **Bahasa Pemrograman:** TypeScript (Untuk kode yang lebih aman dan terstruktur)
4.  **Styling:** CSS3 (Vanilla) dengan CSS Modules untuk skop lokal
5.  **Animasi:** Framer Motion (Disarankan untuk animasi React yang smooth) atau CSS Animations
6.  **Deployment:** Static Hosting (Netlify/Vercel/GitHub Pages)

### Fitur Interaktif (Pure JS)
- Smooth scroll navigation
- Animated counters
- Image lightbox untuk screenshot
- Testimonial carousel/slider
- FAQ accordion
- Sticky navbar with scroll effect
- Mobile responsive hamburger menu
- Floating CTA button
